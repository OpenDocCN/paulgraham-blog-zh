N 事物清单

来源：https://www.yuque.com/hardwaylab/paulgraham/dz3glm
The List of N Things



2009 年 9 月

我敢打赌，当前一期的 Cosmopolitan 有一篇标题以数字开头的文章。 “他不会告诉你的关于性的 7 件事”，或者类似的东西。一些流行杂志在每一期的封面上都刊登了这种类型的文章。这不可能是偶然发生的。编辑必须知道他们会吸引读者。

为什么读者如此喜欢n个事物的列表？主要是因为它比普通文章更容易阅读。 [1] 从结构上讲，n 个事物的列表是一篇退化的文章。一篇文章可以去任何作者想要的地方。在一个包含 n 个事物的列表中，作者同意将自己限制在一组大致相同重要性的点上，并且他明确地告诉读者它们是什么。

阅读一篇文章的一些工作是理解它的结构——弄清楚我们在高中时称之为“大纲”的内容。当然不是明确的，但真正理解一篇文章的人之后可能会在脑海中产生与这样一个大纲相对应的东西。在 n 个事物的列表中，这项工作已为您完成。它的结构是一个外骨骼。

除了明确之外，还保证结构是最简单的类型：几个主要点，很少或没有从属点，并且它们之间没有特定的联系。

因为要点不连通，所以n个事物的列表是随机访问的。没有你必须遵循的推理线索。您可以按任何顺序阅读该列表。因为这些点是相互独立的，所以它们就像一艘不沉船中的防水隔间一样工作。如果您对某一点感到厌烦，或无法理解，或不同意某一点，则不必放弃这篇文章。你可以放弃那个并跳到下一个。 n 个事物的列表是并行的，因此是容错的。

有时这种格式正是作者想要的。一个，很明显，当你要说的实际上是一个包含 n 件事的列表时。我曾经写过一篇关于扼杀初创公司的错误的文章，有些人取笑我写了一些标题以数字开头的东西。但在那种情况下，我真的试图为许多独立的事物制作一个完整的目录。事实上，我试图回答的问题之一是有多少。

使用这种格式还有其他不太合理的原因。例如，我在接近截止日期时使用它。如果我必须发表演讲，而我提前几天还没有开始，我有时会谨慎行事，把演讲列成 n 件事的清单。

n 事物的列表对作者和读者来说都更容易。当你在写一篇真正的文章时，你总是有可能陷入死胡同。一篇真正的文章是一个思路，有些思路会逐渐消失。当您必须在几天内发表演讲时，这是一种令人担忧的可能性。如果你的想法用完了怎么办？ n 事物列表的分隔结构保护作者免于自己的愚蠢，就像保护读者一样。如果您在某一点上没有想法，没问题：它不会杀死文章。如果需要，您可以删除整点，并且文章仍然存在。

写n件事的清单真是太放松了。您在前 5 分钟内想到了其中的 n/2 个。太好了，结构就是这样，你只需要填写它。当你想到更多的点时，你只需将它们添加到最后。也许您取出或重新排列或组合一些，但在每个阶段，您都有一个有效的（尽管最初是低分辨率的）n 个事物的列表。这就像一种编程，你非常快速地编写第 1 版，然后逐渐修改它，但在每一点都有可用的代码——或者你从一个小时内完成的完整但非常模糊的草图开始的绘画风格，然后花费一个星期来解决这个问题。

因为 n 个事物的列表对作家来说也更容易，所以当读者喜欢它时，它并不总是一个诅咒的标志。这并不一定是读者懒惰的证据；这也可能意味着他们对作者没有太大信心。在这方面，n 件事的清单是论文形式的芝士汉堡。如果你在一家你怀疑很糟糕的餐厅吃饭，你最好的选择是点芝士汉堡。即使是一个糟糕的厨师也可以做出一个像样的芝士汉堡。关于芝士汉堡应该是什么样子，有非常严格的约定。你可以假设厨师不会尝试一些奇怪和艺术的东西。 n 件事的列表同样限制了一个糟糕的作家可能造成的损害。你知道这将是关于标题所说的任何内容，并且这种格式可以防止作者沉迷于任何幻想。

因为n个事物的列表是最简单的essay形式，所以对于初学者来说应该是一个不错的选择。事实上，这是大多数初级作家所学的。经典的 5 段文章实际上是 n = 3 的 n 个事物的列表。但是写它们的学生没有意识到他们使用的结构与他们在 Cosmopolitan 中阅读的文章相同。他们不允许包含数字，并且他们应该用无缘无故的过渡（“Furthermore...”）来填补空白，并在两端用介绍性和结束性的段落加盖，这样它表面上看起来就像一篇真正的文章. [2]

让学生从 n 件事的清单开始，这似乎是一个很好的计划。这是最简单的形式。但是，如果我们要这样做，为什么不公开这样做呢？让他们写下 n 个事物的列表，比如专业人士，有数字，没有过渡或“结论”。

在一种情况下，n 个事物的列表是一种不诚实的格式：当您通过错误地声称该列表是详尽无遗的来使用它来吸引注意力时。也就是说，如果你写了一篇声称是关于成功的 7 个秘诀的文章。这种头衔与侦探一样是一种反射性的挑战。您至少必须查看文章以检查它们是否与您列出的 7 个相同。您是否忽略了成功的秘诀之一？最好检查一下。

如果你真的相信你已经列出了一个详尽的清单，那么将“The”放在数字之前是可以的。但有证据表明，大多数带有此类标题的东西都是链接诱饵。

n 事物列表的最大弱点是新思想的空间太小。如果做得好，论文写作的重点是你在写作时的新想法。顾名思义，一篇真正的文章是动态的：开始时你不知道要写什么。它将与您在编写过程中发现的任何内容有关。

这只能在 n 个事物的列表中以非常有限的方式发生。你首先制作标题，这就是它的内容。您在写作中的新想法不能超过最初设置的防水隔间。你的大脑似乎知道这一点：因为你没有新想法的空间，所以你没有它们。

向初学者承认 5 段文章实际上是 n 件事的列表的另一个好处是我们可以警告他们这一点。它只能让您在小范围内体验论文写作的定义特征：一两句话的想法。尤其危险的是，5 段文章将 n 项内容的列表隐藏在看起来更复杂的文章类型中。如果你不知道你在使用这个表单，你就不知道你需要逃避它。

笔记

[1] 这种类型的文章在 Delicious 上也非常受欢迎，但我认为这是因为美味/流行是由书签驱动的，而不是因为 Delicious 用户是愚蠢的。好吃的用户是收藏家，n个东西的列表看起来特别有收藏价值，因为它本身就是一个收藏。

[2] 学校数学教科书中的大多数“单词问题”同样具有误导性。它们表面上看起来像是数学在实际问题中的应用，但事实并非如此。因此，如果有的话，它们会强化这样一种印象，即数学只是需要记住的复杂但毫无意义的东西集合。
