What Microsoft Is this the Altair Basic of?

2015 年 2 月

如果你想了解初创公司，你可以尝试的最有价值的练习之一就是看看最成功的公司，并解释为什么它们不像刚推出时看起来那么蹩脚。因为他们一开始几乎都看起来很蹩脚。不仅小，还跛。不仅仅是上大山的第一步。更像是进入沼泽的第一步。

Altair 的基本解释器？怎么可能成长为一家大公司？人们睡在陌生人公寓的气垫床上？一个供大学生互相跟踪的网站？为使用电视作为显示器的业余爱好者设计的小型单板计算机？一个新的搜索引擎，当已经有大约 10 个的时候，他们都试图淡化搜索？这些想法不仅看起来很小。他们似乎错了。他们是那种你不仅不能忽视，还可以嘲笑的想法。

通常创始人自己不知道为什么他们的想法很有前途。他们本能地被这些想法所吸引，因为他们生活在未来，他们感觉到缺少了一些东西。但是他们无法确切地说出他们的丑小鸭将如何成长为又大又漂亮的天鹅。

大多数人在听到一个听起来很蹩脚的新创业想法时的第一反应就是取笑它。甚至很多应该知道的人。

当我遇到一个听起来很蹩脚的想法的初创公司时，我会问“这是什么 Microsoft 是 Altair Basic 的？”现在这是一个谜题，我的负担是解决它。有时我想不出答案，尤其是当这个想法是虚构的时候。但令人惊讶的是，结果往往是有答案的。通常它是创始人自己还没有见过的。

有趣的是，有时会有多种答案。几天前，我与一家初创公司进行了交谈，该公司可能会成长为 3 个不同的 Microsoft。它们的大小可能会按数量级变化。但是你永远无法预测微软会有多大，所以在这种情况下，我鼓励创始人走最让他们兴奋的道路。他们的直觉让他们走到了这一步。为什么现在停下来？
